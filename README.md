dominator
=========

Distributed application deployment using Docker

Installation
============

Using pip:

`pip install dominator`

On Ubuntu:
```
sudo apt-add-repository ppa:yandex-sysmon/dominator
sudo apt-get update
sudo apt-get install dominator
```
